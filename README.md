# K-Tech The First Project
## 실행
- 본 프로젝트를 clone 받는다.
- Intellij IDEA를 이용해 clone 받은 풀더를 연다.
- Main.java의 main을 실행한다.

## 기술
- Java 8.0 version.
- Java 17.0.9 jdk.

## 이걸 만들면서 고민했던 내용
- 사용자가 어떻게 빠르게 입력할 수 있는지를 고민했다.
- 파일에 데이터를 입출력하는 데 에러 발생하지 않도록 어떻게 처리하야할 지를 
고민했다.
- 사용자로부터 console에서 입력할 때 예외처리 발생을 어떻게
대응해야 할지를 고민했다.
